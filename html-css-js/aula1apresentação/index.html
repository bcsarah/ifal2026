Aprendendo HTML

Introdução

Para criar um arquivo HTML, crie uma pasta e, dentro dele, um arquivo .html. Nesse arquivo, precisamos de um código básico, onde podemos colocá-los diretamente digitando ! e dando enter, se usarmos o VS Code. Desse jeito, temos o código inicial, como esse aqui abaixo.

Com o código básico, nós temos algumas "tags". Essas tags são abertas e fechadas, como o html, que abrimos ele, e fechamos.
